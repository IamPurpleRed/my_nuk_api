My NUK
===
###### tags: `GitHub`, `Node.JS`

## 簡介 | Introduction
一個簡易的API提供給 my_nuk Flutter APP 使用。

A simple API for my_nuk Flutter APP.

## 路由 | Routes
- verification (驗證在APP上的登入資訊是否正確)
- user (提供給使用者的所有功能)
  - grades (查詢歷年學期成績和排名)
  - schedule (查詢本學期課表)
